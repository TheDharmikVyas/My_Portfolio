# Dharmik Vyas | Developer Portfolio

A premium, highly-interactive developer portfolio built from scratch for **Dharmik Vyas**, featuring a sleek monochrome dark design with balanced lemon yellow accents, smooth hover transitions, and responsive mobile layouts.

## 🚀 Features

- **Watermark Hero Title**: Large display heading with a layered, handwritten signature overlay ("Dharmik Vyas") and typewriter subtitle highlights.
- **Concentric Circular Avatar**: Interactive profile avatar section using offset stroke outlines.
- **Academic Milestones**: Clean stats grids highlighting academic CPI and scores.
- **Interactive Project Cards**: Responsive grids featuring project details and technology tags. **Clicking anywhere on a card redirects directly to its GitHub repository.**
- **Project Filter Tabs**: Switch instantly between *All*, *Java Backend*, and *Web & JS* categories.
- **Timeline & Milestones**: Scrollable history tracking academic milestones at DDU and leadership credentials.
- **Let's Talk Contact Footer**: Minimalist messaging form with floating label fields and custom local verification feedback.
- **Performance & SEO**: Built entirely in static vanilla HTML, CSS, and JS with optimized meta tags for instant page load times and zero-cost hosting.

## 🛠️ Technology Stack

- **Core Structure**: Semantic HTML5 & Modern CSS3 Grid/Flexbox
- **Logic & Interactions**: Vanilla JavaScript (ES6+), `IntersectionObserver` scroll reveals
- **Typography (Google Fonts)**: Montserrat (headings), Alex Brush (handwritten script), Plus Jakarta Sans (paragraphs), JetBrains Mono (monospaced tags)
- **Icons**: Inline vector SVGs

## 📂 Project Structure

```bash
dharmik-portfolio/
├── index.html     # Page structure and markup
├── styles.css     # Design tokens, variables, and visual layouts
├── script.js      # Typewriter, filters, reveals, and card clicks
└── profile.jpeg   # Avatar image asset
```

## ⚙️ How to Run Locally

1. Clone or download this directory to your machine.
2. Double-click `index.html` to open the portfolio locally in any modern browser (Chrome, Safari, Edge, Firefox).
